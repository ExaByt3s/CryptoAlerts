## Crypto Alerts
```
Laravel 5.8, PHP7.2, Google App Engine
```
## Project setup

```bash
git clone [this_repo]
composer install
```

## MYSQL Setup using Docker
```
docker-compose up
```

## Run Migrations, Create DB
```
php artisan migrate
```

## Run Application Locally
```
php artisan serve
```

## Deploy to App Engine 
```
gcloud app deploy --project=[your_app_id_here]
```
